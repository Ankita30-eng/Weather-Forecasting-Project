# Weather-Forecasting-Project
This project is a full-stack Python application built using Django as the backend framework. It integrates weather APIs to provide real-time forecasts and displays them through a clean, responsive web interface. The app demonstrates key concepts of full-stack development, including:

Frontend: HTML, CSS, JavaScript for user-friendly design and interactivity.

Backend: Django for handling requests, managing data, and connecting to external weather services.

Database: Storing user queries, preferences, or historical weather data.

API Integration: Fetching live weather information from third-party services.
